# perceptron
Simple perceptron in C

### Example of the results

#### Parameters
100x100 images, sample size - 500, bias - 6.0

#### Results
Accuracy - Untrained model (Law of Large Numbers - Proof): 500/1000 - 50.00%
Accuracy - Trained model: 694/1000 - 69.40%
